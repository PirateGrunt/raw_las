A Simple Plots

plot(x=1,y=1)

Features include:

  • Box around plot area
  • Point is represented by a circle
  • No title or subtitle
  • Default axis labels
  • Axis limits are determined by data

Plot Layout

layout divides the device up into as many rows and columns as there are in matrix, with the column-widths and the row-heights specified in the respective arguments.

layout(matrix(1:4,nrow=2))
layout.show(4)

Plot Layout

You can create your own summary plot for exploratory data analysis.

m1<-matrix(1:4,2,2)
x1<-rnorm(100)
layout(m1)

plot(x1)
hist(x1)
boxplot(x1)
qqnorm(x1)

Plot Layout

Let’s write this as a function.

mysummaryPlot<-function(x){
  m1<-matrix(1:4,2)
    layout(m1)
    plot(x)
    hist(x)
    boxplot(x)
    qqnorm(x)
    }

Titles and Axis Labels

main and sub refer to the main title and subtitle, respectively. Sometimes a plot will have a default title. If you want to supress that you can usemain=NULL and sub=NULL. xlab and ylab are the x-axis and y-axis labels, respectively.

plot(x=1,y=1,xlab="Here's where xlab goes",
  ylab="Here's where ylab goes",type="n",bty="n")
title(main="Here's the main title",sub="Here's the subtitle")

Plotting Functions

There are two ways to plot functions. One is to plot the x and y values and connect the points. The other is to use curve.

f1<-function(x) x*(x^2-1)
x1<-seq(from=-5,to=5,length=100)
y1<-f1(x1)

plot(x=x1,y=y1,type="l")
